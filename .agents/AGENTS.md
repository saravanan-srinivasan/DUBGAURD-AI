# Agent Rules

- NEVER fill the C: drive. Always keep at least 7.5 GB of free space available on the C: drive at any time. When downloading large AI models or dependencies, route them to the D: drive (e.g. `D:\AI_Cache`) to prevent C: drive exhaustion.

- At the START of every new session on this project, ALWAYS read `.agents/PROJECT_MEMORY.md` first before doing anything. This file contains the full project state, all live URLs, deployment instructions, bug history, and critical rules.
